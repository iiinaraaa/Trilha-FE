pseudo-class

Pseudo-classes (:) São palavras-chave que você usa com : para selecionar elementos em um estado específico ou com uma condição especial. Pseudo-elementos (::) São palavras-chave que você usa com :: para estilizar uma parte específica de um elemento — como o primeiro caractere, a primeira linha, ou inserir conteúdo. ex no modulo2>ex6>pseudo-elementos: p:hover { color: aliceblue; } p::before { content: '123'; }

-------------

* = id . = class : = pseudo-class :: = pseudo-elements > = children

BOX MODEL

Todo elemento HTML é tratado como uma caixa retangular. O modelo de caixa define como o espaço é calculado: margin ← espaço externo da caixa border ← borda da caixa padding ← espaço interno, entre a borda e o conteúdo content ← onde o texto/imagem vai
